## Тестовое задание

- **ЯП**: C++

```
Реализовать простую шаблонную циклическую очередь, безопасную для одновременной
записи и чтения из двух потоков (один поток читает, другой пишет), не
использующей механизмы взаимной блокировки потоков.
```

### Требования:

- Память для хранения данных **<ins>аллоцируется статически</ins>** внутри
  очереди.
- Два шаблонных параметра:
  - T – тип данных
  - CAPACITY – ёмкость очереди (максимальное количество хранимых элементов типа
    T).
- От класса T необходимы:
  - реализация конструктора копирования
  - оператора присваивания
- Очередь должна поддерживать:
  - конструктор копирования
  - оператор присваивания
- Объекты:
  - хранятся в очереди до момента, когда они будут «затёрты» новыми данными.
  - Должны уничтожаться **<ins>сразу</ins>** после извлечения из очереди.
